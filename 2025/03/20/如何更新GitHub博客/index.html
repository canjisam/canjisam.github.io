自动化部署

除了手动部署外，还可以设置GitHub Actions实现自动部署。每当你推送更改到GitHub仓库时，GitHub Actions会自动构建并部署你的博客。

常见问题

部署失败

如果部署失败，可能是以下原因：

  1. Git配置问题：确保已正确配置Git用户名和邮箱
  2. SSH密钥问题：确保已添加SSH密钥到GitHub账户
  3. 权限问题：确保有权限推送到目标仓库
